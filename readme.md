# Exabase Examples

This repository contains examples of how to use Exabase with different use cases.

## Examples

- [AI Flashcard generator](ai-flashcard-generator): A simple example of an AI flashcard generator using Exabase, Next.js, AI SDK, and Tailwind CSS.

# What can you do with Exabase?

- Analyze files with AI to provide summary, metadata, captions
- Store/retrieve memories
- Search anything stored semantically
